API INTEGRATION

RESTful API with comprehensive endpoints for seamless integration

Quick Start

# Install dependencies
pip install flask scikit-learn pandas numpy

# Run the API server
python app.py

# Server will start on http://localhost:5000

Example Request

import requests

# Prepare customer data
data = {
    "credit_score": 650,
    "age": 35,
    "tenure": 5,
    "balance": 50000,
    "num_products": 2,
    "has_cr_card": 1,
    "is_active_member": 1
}

# Make prediction request
response = requests.post(
    "http://localhost:5000/api/predict",
    json=data
)

result = response.json()
print(f"Churn Probability: {result['probability']}%")
print(f"Risk Level: {result['risk_level']}")

API Endpoints

POST
/api/predict

Get churn prediction for a single customer with probability score and risk level.

POST
/api/batch-predict

Process multiple customer records in a single request for bulk predictions.

GET
/api/models

Retrieve performance metrics for all available ML models in the ensemble.

GET
/api/feature-importance

Get feature importance scores to understand which factors drive predictions.
